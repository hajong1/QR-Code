얼마전 큐싱이라는 사이버 범죄가 대두된 것을 보고 어떻게 예방할 수 있을지 고민하게 되었습니다.
일상화되어 있는 QR코드를 안심하고 이용 할 수 있는 날을 기대하면서 QR 코드 스캐너를 만들게 되었습니다.

- 심플하게 QR코드를 스캔할 수 있습니다.
- 스캔했던 QR코드는 나만 볼 수 있게 기록됩니다.
- 위험 할 수 있는 QR코드를 감지해 알립니다 (개발예정)
